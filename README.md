# 맞춤법 교정 챗봇 
React + AWS 맞춤법 교정 챗봇

## 프로젝트 소개
hanspell 라이브러리를 활용하여 만든 맞춤법 교정 챗봇입니다.

## 개발기간
* 23.11.20~ 23.12.15

## 개발환경
- AWS Lambda
- React.js
- haspell(node.js)
